ls command - List directory contents

ls
ls -lrash

Some useful options are -l, -a, -s, -h and -R where -l will give you a long listing (as explained above) -a will show you ALL the files in the directory, including hidden files -R will the subdirectories recursively, which means it will show all the directories and files within the specified directory. -s will also show you the size of the files (in blocks, not bytes) -h will show the size in "human readable format" (ie: 4K, 16M, 1G etc). Of course you must use this option in conjunction with the -s option.

Create or Extract tar file

-Create

tar -cvf tarfilename.tar /path/to/directory  // Crate tar of give driectory
tar -cvf tarfilename.tar /path/to/file-1 /path/to/file-2 //To create an archive of certfain files
tar -cvf tarfilename.tar /path/to/directory --exclude="*.png" //To exclude certian file from tar.

-Extract

tar -xvf tar_file.tar
tar -xvf tar_file.tar note.txt //Extract single file from tar
tar -xvf tar_file.tar bin // Extract a single directory called bin,

-To see file and directory in tar without extract

tar -tf tar_filename.tar

chmod command - To change permission

-chmod u=rwx,g=rx,o=r myfile -Where, The letters u, g, and o stand for "user", "group", and "other" and the letters "r", "w", and "x" stand for "read", "write", and "execute".

  • Each digit is a combination of the numbers 4, 2, 1, and 0. --4 stands for "read", --2 stands for "write", --1 stands for "execute", and --0 stands for "no permission." -So 750 mean U- (4+2+1)rwx, G- (0+4+1)rx, O-(0+0+0) NO permission
chmod 755 ./temp
